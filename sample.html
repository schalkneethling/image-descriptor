Instructions

For Generating Alt Text:

  1. Place your cursor inside any of the empty alt="" attributes above
  2. Right-click to open the context menu
  3. Select "Image Descriptor: Suggest alternative text"
  4. The extension will generate and insert appropriate alt text

For Translating Alt Text:

  1. Place your cursor inside any of the Spanish alt="" attributes above
  2. Right-click to open the context menu
  3. Select "Image Descriptor: Translate to English"
  4. The extension will translate the Spanish alt text to English while preserving context and nuance
